The latest version of @tokens-studio/sd-transforms does not export registerTransformGroup. It now uses a default export function that registers all necessary transforms when invoked.

Replace this code:

import { registerTransformGroup } from ‘@tokens-studio/sd-transforms’
registerTransformGroup(StyleDictionary)

With this:

import tokensStudioTransforms from ‘@tokens-studio/sd-transforms’
tokensStudioTransforms(StyleDictionary)

This will correctly register the tokens-studio transform group with Style Dictionary v5. No other changes are needed. Continue using StyleDictionary.extend and buildAllPlatforms as before.
