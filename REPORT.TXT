### **Put your token file in the right place**

Move your DDS Foundations.json here:

```bash
ai-design-system/tokens/foundations/DDS Foundations.json
```

### **Create this config file**

Make a file at:
ai-design-system/config/config.json

Paste this inside:

```json
{
  "source": ["tokens/foundations/DDS Foundations.json"],
  "platforms": {
    "css": {
      "transformGroup": "tokens-studio",
      "buildPath": "build/css/",
      "files": [
        {
          "destination": "variables.css",
          "format": "css/variables"
        }
      ]
    }
  }
}
```

### **Then run:**

```bash
npm install style-dictionary @tokens-studio/sd-transforms --save-dev
npx style-dictionary build --config config/config.json
```

This will spit out a build/css/variables.css file with all your primitives.

# Style Dictionary Quick Proof of Concept

## Current State
We have Style Dictionary configuration in two locations:
- `/config` with transforms and transformGroups
- `/style-dictionary` with style-dictionary.config.mjs

## Immediate Steps (5 minutes)

1. **Consolidate into style-dictionary directory**
   ```bash
   # Create structure inside style-dictionary
   mkdir -p style-dictionary/tokens/foundations
   mkdir -p style-dictionary/build/css
   
   # Move ALL config into style-dictionary
   mv config/* style-dictionary/
   rm -rf config
   ```

2. **Move your token file**
   ```bash
   cp "token-studio-sync-provider/DDS Foundations.json" "style-dictionary/tokens/foundations/DDS Foundations.json"
   ```

3. **Update config**
   ```bash
   cat > style-dictionary/style-dictionary.config.mjs << 'EOL'
   import StyleDictionary from 'style-dictionary';
   import * as sdTransforms from '@tokens-studio/sd-transforms';

   // Register all tokens-studio transforms and formats
   sdTransforms.registerWith(StyleDictionary);

   export default {
     source: ['tokens/foundations/DDS Foundations.json'],
     platforms: {
       css: {
         transformGroup: 'tokens-studio',
         buildPath: 'build/css/',
         files: [{
           destination: 'variables.css',
           format: 'css/variables'
         }]
       }
     }
   };
   EOL
   ```

4. **Install and run (one command)**
   ```bash
   cd style-dictionary && npm install style-dictionary @tokens-studio/sd-transforms --save-dev && npx style-dictionary build
   ```

## Expected Results
- A `style-dictionary/build/css/variables.css` file containing your tokens as CSS variables
- All DTCG tokens properly transformed
- Ready to use in your project
